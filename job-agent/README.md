# 🤖 Job Agent — Daily AI-Powered Job Digest

Scrapes LinkedIn & Naukri for relevant jobs, scores them against your profile, and emails a digest every morning.

---

## How it works

```
Step 1 — Setup (once)
  python setup_profile.py resume.pdf
       │
       ▼
  Reads PDF → Claude AI extracts profile → writes config/profile.py
       │
       ▼
Step 2 — Run agent (daily)
  python scheduler.py
       │
       ├── Scrape LinkedIn (JSearch API)
       ├── Scrape Naukri (web)
       ├── Filter & score by profile
       ├── Deduplicate (skip already-sent jobs)
       └── Email HTML digest
```

---

## Quick Start

### 1. Install
```bash
pip install -r requirements.txt
```

### 2. Set credentials
```bash
cp .env.example .env
# Edit .env — add ANTHROPIC_API_KEY, GMAIL_APP_PASSWORD, RAPIDAPI_KEY
```

### 3. Generate your profile from resume (first-time setup)
```bash
python setup_profile.py path/to/your_resume.pdf
```
This reads your PDF, calls Claude AI, and writes `config/profile.py` automatically.
Takes ~15 seconds. Run again anytime you update your resume.

### 4. Test the agent
```bash
python scheduler.py --now
python scheduler.py --now --fresh
```

### 5. Start the daily scheduler
```bash
python scheduler.py        # Runs every day at 08:00 AM
```

> **Note:** If `config/profile.py` is missing or unconfigured, the scheduler will block and remind you to run `setup_profile.py` first.

## Re-running setup for a different resume

Just run setup again — the old `profile.py` is automatically backed up:
```bash
python setup_profile.py new_resume.pdf
# Backs up old profile to config/profile_backup.py
# Writes new config/profile.py
```

## Project structure

```
job-agent/
├── setup_profile.py          ← Run this first (resume → profile.py)
├── scheduler.py              ← Run this daily (guarded by profile check)
├── agents/
│   └── job_agent.py          ← Orchestrator
├── scrapers/
│   ├── linkedin_scraper.py   ← LinkedIn via JSearch RapidAPI
│   └── naukri_scraper.py     ← Naukri web scraper
├── utils/
│   ├── job_filter.py         ← Filter + relevance scoring
│   ├── email_sender.py       ← Gmail SMTP + HTML template
│   └── deduplicator.py       ← Tracks sent jobs
├── config/
│   ├── profile.py            ← Auto-generated by setup_profile.py
│   └── settings.py           ← Reads from .env
├── data/seen_jobs.json       ← Auto-created
├── logs/agent.log            ← Auto-created
├── requirements.txt
└── .env.example
```

## Deploy options

### Cron (Linux/Mac)
```bash
crontab -e
0 8 * * * cd /path/to/job-agent && python scheduler.py --now >> logs/cron.log 2>&1
```

### GitHub Actions (free, no server)
```yaml
name: Daily Job Digest
on:
  schedule:
    - cron: '30 2 * * *'   # 8 AM IST = 2:30 AM UTC
jobs:
  run:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: pip install -r requirements.txt
      - run: python scheduler.py --now
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          GMAIL_USER: ${{ secrets.GMAIL_USER }}
          GMAIL_APP_PASSWORD: ${{ secrets.GMAIL_APP_PASSWORD }}
          RAPIDAPI_KEY: ${{ secrets.RAPIDAPI_KEY }}
```
